# Requirement (validated version)
  
- Python 3.x (3.7.8)
 - matplotlib (3.3.1)
 - pyEMMA (2.5.7)
 - numpy (1.19.1)
 - tqdm (4.48.2)
 
# Installation
Launch string.ipynb and load post-string-input.xvg.
post-string-input.xvg should contain three column data (x, y, weight).
